
Kiezatlas Wordpress Plugin
==========================

## Nutzungshinweise

- Den `kiezatlas-page`-Ordner in den `wp-content/plugins/`-Ordner kopieren
- Der Stadtplan muss unter `/kiezatlas-citymap/stadtplan` veröffentlicht sein
